# 项目结构
  - spas-convert-core: 项目工具集
  - spas-convert-sample: 项目示例(包含mapstruct实体映射框架)
  - spas-convert-sdk: 项目核心业务逻辑
  - spas-convert-spring-boot-starter: 项目web服务,端口: 9999
  - spas-convert-swing: 项目界面化配置
  - spas-dependencies: 项目版本引用
